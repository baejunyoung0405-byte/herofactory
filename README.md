# HeroFactory MVP

탑뷰 형식 던전 게임의 웹 MVP입니다.

## 실행
1. `herofactory.html`을 브라우저에서 열면 바로 실행됩니다.
2. 또는 간단한 정적 서버를 사용해 실행할 수 있습니다.

## 구현된 기능
- 시작 화면: `출격`, `기지`, `설정`, `종료`
- 출격:
  - 행성 선택 모달
  - 마우스 드래그로 행성 맵 이동
  - 행성 hover 시 테두리 강조 + 우측 정보 패널 갱신
  - 행성/지역 선택
- 전투:
  - 간단한 탑뷰 전투(이동: `WASD`, 자동 공격)
  - 클리어 시 지역 `수복`
  - 수복률에 따른 적 약화 반영
- 기지:
  - 본부/공장/발전소/마켓/블랙 마켓/격납고/수감소 탭
  - 마켓 용병 계약, 해지 시 계약금 80% 환급
- 설정:
  - 사운드, 언어, 난이도, 무적, 무한 재화
- 저장:
  - `localStorage` 자동/수동 저장

## 파일 구성
- `herofactory.html`: 화면 구조
- `styles.css`: UI 스타일
- `game.js`: 상태/전투/기지/수복/저장 로직
